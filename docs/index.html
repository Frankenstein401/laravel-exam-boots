Instalasi & Pengenalan

Laravel Exam Boots menggunakan konsep ejection (shadcn/ui philosophy). Kode-kode generator langsung diletakkan di dalam folder `app/` agar kamu memiliki kontrol 100% saat kustomisasi soal ujian.

Langkah Instalasi

  1. Jalankan composer require di root project Laravel kamu:
    composer require gani/laravel-exam-boots
  2. Pastikan package ter-load dengan benar:
    php artisan exam

Pre-flight Environment Doctor

Sebelum mulai ujian, pastikan lingkungan development kamu sudah siap dengan perintah `exam:doctor`. Perintah ini meminimalkan potensi kegagalan di tengah jalan akibat masalah konfigurasi DB atau PHP.

Artisan CRUD Generator

Buat file scaffolding lengkap (Model, Migration, Controller, Service, Request, Resource) hanya dengan satu command:

Scaffolding Command
php artisan exam:add Product

Flag Options yang Sangat Penting

--belongsTo=Category

Otomatis menyuntikkan relasi belongsTo di model saat ini, hasMany di model Parent, serta kolom Foreign Key di migration.

--with-factory

Membuat file Model Factory & Database Seeder sekaligus, terintegrasi otomatis.

--upload=image

Menyisipkan handler upload file (image/pdf/etc.), validasi ukuran file, logic Storage, dan property resource URL.

--web

Mengaktifkan mode Web MVC: Menghasilkan 4 file Blade views (index, create, edit, show) siap pakai yang di-style dengan Tailwind CSS, Controller redirect, dan auto-register ke routes/web.php.

--enum=status:active,inactive

Membuat file Enum baru (app/Enums/Status.php), menambahkan casting Enum ke Model, dan menyiapkan tipe data ENUM di database migration.

--soft-deletes

Menyisipkan trait SoftDeletes ke Model dan mendefinisikan kolom softDeletes() pada file migration database.

--dry-run / --force

--dry-run digunakan untuk preview operasi file sebelum dieksekusi. --force digunakan untuk menimpa file yang sudah ada tanpa konfirmasi prompt.

Setup Authentication

Buat sistem register, login, profile info (me), dan logout instan. Kamu hanya perlu memilih driver token auth:

Langkah Setup
  1. Jalankan: php artisan exam:auth
  2. Pilih driver: **JWT** (tymon/jwt-auth) atau **Sanctum** (laravel/sanctum).
  3. Ketik **Yes/Ya** jika ingin sekaligus di-instalkan **API docs Scramble** secara otomatis.

Scramble API Docs

Scramble mendokumentasikan API Laravel secara dinamis tanpa anotasi PHPDoc yang rumit. Jika diaktifkan saat setup auth, program akan meng-configure Service Provider agar mendeteksi metode autentikasi secara otomatis.

📍 URL Dokumentasi API dapat diakses di: http://localhost:8000/docs/api setelah server berjalan.

Relasi Interaktif

Gunakan `exam:relation` untuk menghubungkan dua model yang sudah ada dengan panduan interaktif terminal. Mendukung relasi **One to One**, **One to Many**, dan **Many to Many**.

CLI Command
php artisan exam:relation

ApiResponse Trait

Standardisasi format JSON API dengan ejecting trait `ApiResponse.php` ke folder `app/Traits/`.

CLI Command
php artisan exam:response
app/Traits/ApiResponse.php

// Di Controller kamu:

use App\Traits\ApiResponse;


class ProductController extends Controller

{

use ApiResponse;


public function index()

{

return $this->successResponse($products, "Data berhasil diload");

}

}

Policy/Authorization

Gunakan `exam:policy {Model}` untuk membuat authorization Policy dan auto-register ke dalam array `$policies` di `AuthServiceProvider.php`.

CLI Command
php artisan exam:policy Product

Feature Test Skeleton

Buat pengujian otomatis untuk modul CRUD dan endpoint protection. CLI mendukung framework testing bawaan PHPUnit atau Pest PHP.

Generate Command
php artisan exam:test Product # PHPUnit
php artisan exam:test Product --pest # Pest PHP

Ekspor Data (Excel/PDF)

Generate boilerplate logic untuk mengekspor data ke file Excel atau PDF lengkap dengan blade layout styling.

CLI Command
php artisan exam:export Product

Rollback & Revert (exam:undo)

Salah ketik nama model atau relasi? Gunakan `exam:undo` untuk mengembalikan file yang dimodifikasi ke versi backup aslinya dan menghapus file yang baru digenerate secara aman.

CLI Command
php artisan exam:undo

Admin User Seeder

Buat data dummy admin default secara instan untuk mempercepat proses testing login API/Web. Command ini men-generate class seeder dan mendaftarkannya langsung ke kelas `DatabaseSeeder.php`.

CLI Command
php artisan exam:seed-admin --email=admin@example.com --password=password
